# Авто-тесты для Selectel

## Реализованные проверки
1. Проверка открытия главной страницы.
2. Проверка отображения основных элементов шапки.
3. Проверка перехода в английскую версию сайта.
4. Проверка наличия ссылки на статус сервисов.
5. Проверка открытия контактов.

## Как запустить локально


### 1. Создать виртуальное окружение
python -m venv venv
venv\Scripts\activate
### 2. Установить зависимости
pip install -r requirements.txt
### 3. Установить браузеры Playwright (можно только один для поверхностной проверки)
playwright install
### 4. Запустить тесты
pytest -v
